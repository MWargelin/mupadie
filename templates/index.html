Mupadie

Welcome! Mupadie is an application to test and evaluate musical pattern discovery algorithms. Start by uploading a piece of music as a MIDI file with the form above to visualise musical patterns in that piece.

You can also upload your own musical pattern discovery results as a JSON file, if you want to visualise patterns found by an algorithm not included in this application! See further instructions uploading JSON files.
